## Laporan Program LatihanCounter

### Deskripsi Program

Program ini dibuat untuk memahami penggunaan operator aritmatika sederhana pada sebuah variabel dalam bahasa pemrograman Java. Program akan memodifikasi nilai sebuah variabel bernama `counter` dan menampilkan setiap perubahan nilainya ke layar.

### Penjelasan Variabel

Program menggunakan satu variabel utama, yaitu:

* **int** → digunakan untuk menyimpan nilai `counter` karena berupa bilangan bulat dan tidak memiliki desimal.

### Penjelasan Operator yang Digunakan

Beberapa operator yang digunakan dalam program ini antara lain:

* **++ (increment)** → menambah nilai variabel sebesar 1.
* **-- (decrement)** → mengurangi nilai variabel sebesar 1.
* **+=** → menambahkan nilai tertentu ke variabel.
* **-=** → mengurangi nilai tertentu dari variabel.
* ***=** → mengalikan nilai variabel dengan angka tertentu.

### Alur Program

1. Program mendeklarasikan variabel `counter` dengan nilai awal 10.
2. Program menampilkan judul "COUNTER" dan nilai awal variabel.
3. Program melakukan operasi increment sebanyak dua kali.
4. Program melakukan operasi decrement sebanyak satu kali.
5. Program menambahkan nilai 5 menggunakan operator `+=`.
6. Program mengurangi nilai 3 menggunakan operator `-=`.
7. Program mengalikan nilai dengan 2 menggunakan operator `*=`.
8. Setiap perubahan nilai langsung ditampilkan menggunakan `System.out.println()`.

### Output Program

Program akan menampilkan nilai `counter` yang berubah secara bertahap setelah setiap operasi dijalankan, sehingga pengguna bisa melihat proses perubahan nilai dari awal sampai akhir.

### Kesimpulan

Program ini membantu memahami konsep dasar perubahan nilai variabel dalam Java menggunakan operator increment, decrement, dan assignment. Materi seperti ini penting sebagai pondasi sebelum masuk ke percabangan atau perulangan karena hampir semua logika program nantinya melibatkan manipulasi nilai variabel.
