# Laporan Program LatihPerhitungan

## Deskripsi Program
Program ini dibuat untuk mensimulasikan proses transaksi sederhana pada saldo rekening, yaitu proses setoran dan penarikan uang. Program akan menghitung saldo akhir setelah dilakukan transaksi.

## Penjelasan Variabel

Beberapa variabel yang digunakan dalam program:

- **int saldo** → menyimpan saldo awal sebesar 1.000.000.
- **int setoran** → menyimpan jumlah uang yang disetorkan sebesar 500.000.
- **int penarikan** → menyimpan jumlah uang yang ditarik sebesar 250.000.
- **int SaldoAkhir** → digunakan untuk menyimpan hasil perhitungan saldo setelah transaksi.

Semua variabel menggunakan tipe data `int` karena nilai yang digunakan berupa bilangan bulat.

## Alur Program

1. Program menampilkan saldo awal.
2. Program menampilkan jumlah setoran.
3. Program menghitung saldo setelah setoran (saldo + setoran).
4. Program menampilkan jumlah penarikan.
5. Program menghitung saldo akhir (saldo setelah setor - penarikan).
6. Program menampilkan saldo akhir.

## Proses Perhitungan

- Saldo awal = 1.000.000  
- Setelah setor = 1.000.000 + 500.000 = 1.500.000  
- Saldo akhir = 1.500.000 - 250.000 = 1.250.000  

## Kesimpulan

Program ini menunjukkan penggunaan operasi aritmatika dasar dalam Java, yaitu penjumlahan dan pengurangan, serta penggunaan variabel untuk menyimpan hasil perhitungan sementara dan akhir.
