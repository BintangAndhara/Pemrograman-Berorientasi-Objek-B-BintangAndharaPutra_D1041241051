# Laporan Program LatihDataMahasiswa

## Deskripsi Program
Program ini dibuat untuk menampilkan data mahasiswa ke layar menggunakan bahasa pemrograman Java. Data yang ditampilkan meliputi NIM, nama, kelas, status keaktifan, IPK semester, dan total SKS.

## Penjelasan Variabel

Program menggunakan beberapa tipe data dasar dalam Java, yaitu:

- **long** → digunakan untuk menyimpan NIM karena nilainya besar.
- **String** → digunakan untuk menyimpan nama mahasiswa dalam bentuk teks.
- **char** → digunakan untuk menyimpan kelas karena hanya terdiri dari satu karakter.
- **boolean** → digunakan untuk menunjukkan status aktif mahasiswa (true = aktif, false = tidak aktif).
- **double** → digunakan untuk menyimpan IPK karena memiliki nilai desimal.
- **int** → digunakan untuk menyimpan total SKS karena berupa bilangan bulat.

## Alur Program

1. Program mendeklarasikan dan menginisialisasi semua variabel.
2. Program menampilkan judul "Data Mahasiswa".
3. Program mencetak seluruh data mahasiswa menggunakan `System.out.println()`.

## Output Program

Program akan menampilkan data mahasiswa sesuai dengan nilai yang telah diinput ke dalam variabel.

## Kesimpulan

Program ini menunjukkan penggunaan berbagai tipe data dasar dalam Java serta cara menampilkan data ke layar menggunakan
